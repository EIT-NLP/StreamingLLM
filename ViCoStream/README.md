## TL;DR
**ViCoStream turns streaming VideoLLM inference into a bounded, coordinated pipeline. By combining chunk-wise processing, temporal visual-token compression, local visual attention, and query-aware memory retrieval, it sustains real-time video ingestion while keeping query latency low and accuracy largely preserved.**
